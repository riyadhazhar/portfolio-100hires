### 3. Challenges & Resolutions
- **Challenge:** Upon the initial launch of the latest Cursor build, the application defaulted to the Agent Window UI. This interface abstracts away the standard extension marketplace, making it unclear how to install third-party tools.
- **Resolution:** I analyzed the interface and identified the option to toggle the view. By clicking the "Editor Window ↗" action in the top navigation bar, I reverted the layout to the classic IDE view. This restored access to the standard Extensions panel, allowing me to complete the required installations without further delays.
